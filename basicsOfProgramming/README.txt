!!!Перед запуском проекта, требуется установить все необходимые библиотеки!!!

1. Разархивируйте файл basicsOfProgramming.rar
2. Откройте папку в среде разработки
3. Откройте терминал и пропишите : \Scripts\activate
4. Для установки всех библиотек пропишите в терминале: 
pip install -r requirements.txt
